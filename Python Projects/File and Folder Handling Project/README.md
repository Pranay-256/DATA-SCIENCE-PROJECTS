📂 File and Folder Handling Project (Python)

A Python-based File and Folder Handling System that demonstrates how to perform common file system operations using Python.

This project is divided into two parts:

1️⃣ Basic File Handling

2️⃣ Advanced File & Folder Handling

The project uses Python modules like Pathlib, OS, and Shutil to perform file operations such as creating, reading, updating, deleting files, and managing folders.

---

📌 Features

📄 Basic File Handling

- Create a new file
- Read an existing file
- Update a file
  - Rename file
  - Overwrite file content
  - Append new data
- Delete files
- View all files and folders in the directory

---

📁 Advanced File & Folder Handling

- View parent folders
- View all folders in the directory
- Create folders
  - Parent folder
  - Child folder
  - Grandchild folder
- Add text files inside folders
- Delete text files
- Read text files
- Rename files and folders
- Delete folders with all contents
