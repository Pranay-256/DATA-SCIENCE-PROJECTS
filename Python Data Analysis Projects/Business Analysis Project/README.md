# 📊 Business Sales Analysis

An end-to-end **Exploratory Data Analysis (EDA)** project that analyzes business sales data to identify revenue patterns, product performance, and category contributions.

This project demonstrates **data cleaning, statistical analysis, visualization, and business insight generation using Python**.

---

# 🚀 Project Objectives

The main objectives of this project are:

- Clean and preprocess messy sales data
- Perform exploratory data analysis (EDA)
- Identify high performing products and categories
- Understand revenue drivers
- Analyze correlations between business metrics
- Generate actionable business insights

---

# 🛠️ Tech Stack

**Programming Language**

- Python

**Libraries Used**

- Pandas
- NumPy
- Matplotlib
- Seaborn

**Environment**

- Jupyter Notebook

---

# 📂 Dataset Description

The dataset contains **sales transaction records**.

| Column | Description |
|------|------|
| Order_ID | Unique order identifier |
| Product | Product name |
| Category | Product category |
| Quantity | Number of units sold |
| Price_per_Unit | Price of a single unit |
| Total_Sale | Total revenue generated |

Dataset Size:

- **Rows:** 100
- **Columns:** 6

---

# 🔎 Project Workflow

```
Raw Dataset
     ↓
Data Cleaning
     ↓
Exploratory Data Analysis
     ↓
Correlation Analysis
     ↓
Distribution Analysis
     ↓
Feature Engineering
     ↓
Business Insights
```

---

# 🧹 Data Cleaning

Steps performed:

- Dataset inspection using `df.info()`
- Missing value detection
- Validation of Product-Category mapping
- Handling unknown values
- Data consistency checks

---

# 📈 Category Level Analysis

Metrics analyzed:

- Average Price per Category
- Total Quantity Sold
- Total Revenue Generated

Visualization used:

- **Bar Chart — Category Level Sales Performance**

### Insight

The **Accessories category contributes the highest revenue**, making it the most important category in the dataset.

---

# 🛍️ Product Level Analysis

Metrics analyzed:

- Total quantity sold per product
- Total revenue generated by each product

Visualization used:

- **Product Sales Bar Chart**

### Insight

The **top performing products are:**

1. Laptop
2. Headphones

These products generate the majority of total revenue.

---

# 🔗 Correlation Analysis

Correlation matrix calculated between:

- Quantity
- Price_per_Unit
- Total_Sale

### Key Findings

- Strong positive correlation between **Quantity and Total Sales**
- Moderate correlation between **Price per Unit and Total Sales**

### Visualizations

Scatter plots showing relationships between:

- Total Sales vs Quantity
- Total Sales vs Price per Unit

---

# 📊 Revenue Contribution Analysis

Revenue contribution was analyzed at two levels.

### Category Contribution

Visualization:

- **Pie Chart**

Result:

- Accessories → Highest contribution
- Electronics → Moderate contribution
- Unknown → Lowest contribution

### Product Contribution

Products generating the most revenue:

- Laptop
- Headphones
- Tablet

---

# 📉 Distribution Analysis

Distribution of **Price_per_Unit** was studied using:

- Descriptive statistics
- Skewness calculation
- Histogram
- IQR based outlier detection

### Findings

- Price distribution is **approximately symmetric**
- No significant outliers were detected
- Most prices fall in the **mid-range segment**

---

# ⚙️ Feature Engineering

Two additional features were created.

### Price Range

Products were categorized into:

- Low
- Medium
- High

### Revenue Bucket

Orders categorized into:

- Low Revenue
- Medium Revenue
- High Revenue

This helps understand how **pricing affects revenue generation**.

---

# 📊 Key Business Insights

✔ Accessories category is the **primary revenue driver**

✔ Laptop and Headphones are the **highest performing products**

✔ Increasing product quantity significantly increases **total revenue**

✔ Revenue distribution is **not equal across categories**

✔ Electronics category shows **potential growth opportunity**

✔ Low price products consistently generate **lower revenue**
